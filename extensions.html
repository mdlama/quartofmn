Useful Extensions

HTML

HTML
Name Description
bluesky-comments Quarto Shortcode Extension to Embed Comments from Bluesky and add Bluesky-powered comment sections for websites and HTML outputs
elevator Shortcode for using Elevator.js in HTML documents
embedio Variety of shortcodes for including files and content in HTML documents
lightbox Create lightbox treatments for images in your HTML documents (see also the built-in functionality)
modal A Quarto extension providing a simple way to create Bootstrap modals in your HTML documents
nutshell Embed Nutshell expandable explanations in HTML documents
remember An extension for Quarto to save and restore navigation position across page visits for both HTML documents and Reveal.js presentations
social-embeds Embed content from across the web into HTML documents using a shortcode
social-share Add buttons to share articles on various social media platforms

Coding

Coding
Name Description
code-appendix A lua filter to auto-generate an appendix from code chunks
output-styling Style computational output, errors, warnings, and messages in HTML documents
pyodide Embed Pyodide in HTML documents to enable interactive Python code cells without the need for a Python server
quarto-live WebAssembly powered ‘live’ formats for Quarto HTML documents and Revealjs slides, adding interactive code blocks and exercises for R and Python
qreacto Run standalone React components in your project
regurgitate Automatic Code Appendix containing all shown code with an option to include results or group by language
shinylive This extension lets you embed Shinylive applications in a Quarto document.
webr Embed WebR in HTML documents to enable interactive R code cells without the need for an R server

Teaching

Teaching
Name Description
assign A Quarto Extension for educators writing assignments by combining instruction and solution authoring (Note: Some if this functionality is built-in to Quarto through Conditional Content)
honeypot Add hidden instructions to HTML homework assignments to help detect cheating by unauthorized LLM usage
quizdown Add an interactive quiz into your HTML document

General Utility

General Utility
Name Description
linkate Automatically collect all URLs in a document and place them at the end under a “Links” section
molstar Shortcode to embed proteins and trajectories with Mol*
open-social-comments Enables adding Bluesky or Mastodon/Fediverse powered comments to posts and articles
unsplash Shortcode to use Unsplash images within documents

Math/LaTeX

Math/LaTeX
Name Description
bibentry A Quarto extension that replicates LaTeX’s \bibentry command functionality. It allows you to display full bibliography entries inline within your document text
callouty-theorem Wraps theorems and proofs in callout blocks for better visual appeal
custom-amsthm-environments Quarto extension for defining custom amsthm environments
imagify Convert embedded LaTeX to images and use .tex/.tikz files as image sources
latex-environment Output divs as custom LaTeX environments
tikz A filter that renders PGF/TikZ diagrams in HTML as SVG
typst-math Changes math block syntax from LaTeX to Typst math
